API

Methods

Change Order

row.reverseOrder() --------------- Reverses the order of elements

row.changeOrder('list') ---------- Changes order of elements based on a list

Add Background

row.addBackgroundImg('item number', 'url') ---------- Adds background image to specified item in row

row.addBackgroundImgFix('item number') -------------- Fixes background image on scroll

row.addBackgroundColor('item number', 'color') ------ Changes colour of a single item in row

row.addBackgroundColorAll('color') ------------------ Changes colour of all items in the row

Change Dimensions

row.setRowWidth('width') ---------- Sets entire row width

row.changeWidths('list') ---------- Changes item width ratios based on list of ratios passed in

row.changeHeights('list') --------- Changes item height ratios based on list of ratios passed in

Alighnment

row.alignRow('direction') --------- Moves entire row to left or right based on direction specified

row.changeAligment('type') --------- Aligns row elements to top, bottom, or center based on type specified

Margins

row.removeMargins() ------------------ Removes all margins

row.changeXMargins('margin') --------- Changes margins on the left and right

row.changeYMargins('margin') --------- Change margins on the top

Height Slider

row.addHeightSlider() ------------------ Adds height slider to specified row

row.setMinMaxHeight('min', 'max') ------ Changes min and max values for the height slider

Add Hover

row.addHoverAnimation('type') --------- Adds hover effect based on type

Border Radius

row.changeBorderRadius('radius') --------- Changes items border radius

Complete Configuration

var row = new Row(4, "35vh")

row.changeBorderRadius(0)

row.changeAligment("bottom")

row.addHoverAnimation("height");
